HTML

HTML is not a programming language. It does not have logic or decision-making. Instead, it is a markup language, which means it is used to mark the structure of the content.

CSS

CSS stands for Cascading Style Sheets. It is the language used to style and design the appearance of HTML content on a webpage.

JS

JavaScript is a programming language used to make web pages interactive and dynamic. While HTML builds the structure and CSS styles the content, JavaScript adds behavior to the page.
